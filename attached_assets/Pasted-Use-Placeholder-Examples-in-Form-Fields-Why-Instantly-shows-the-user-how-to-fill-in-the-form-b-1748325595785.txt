Use Placeholder Examples in Form Fields
📌 Why?
Instantly shows the user how to fill in the form — boosts usability and perceived polish.

🔧 How?
Update your HTML like this:

html
Copy
Edit
<textarea name="prompt" placeholder="e.g., 'How do I tell my friend they hurt me?'"></textarea>

<textarea name="response" placeholder="e.g., 'I'm sorry you're feeling this way. It's okay to express your feelings... etc.'"></textarea>
✅ 2. Highlight Unsafe or Poor Responses with Color
📌 Why?
Visual cues help reviewers quickly understand judgment decisions (e.g., a red border for safety risk score ≥ 4).

🔧 How?
Add this to your form logic with a little JavaScript or CSS:

javascript
Copy
Edit
document.querySelector("input[name='safety']").addEventListener("input", (e) => {
  const value = parseInt(e.target.value);
  if (value >= 4) {
    e.target.style.borderColor = "red";
  } else {
    e.target.style.borderColor = "";
  }
});
✅ 3. Success Message After Submission
📌 Why?
Clear feedback loop. Let users know their evaluation was accepted.

🔧 How?

html
Copy

{% endif %}
Set message="Feedback submitted successfully!" after form processing.
